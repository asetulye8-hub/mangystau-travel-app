# Туристік қосымша (Tourist App)

## Сипаттама
Бұл қосымша пайдаланушыларға туристік орындарды қосуға, көруге, өңдеуге және жоюға мүмкіндік береді.  
Пайдаланушылар тіркеліп, өз орындарын сақтай алады. Қосымша офлайн және онлайн режимде жұмыс істейді.

### Негізгі мүмкіндіктер:
- Пайдаланушы тіркелуі және кіруі (Firebase Authentication арқылы)
- Орындарды сақтау және оқу (Room + Firebase Realtime Database)
- Орындарды қосу, өңдеу және жою
- Оффлайн режимде деректерге қолжетімділік

## Технологиялар
| Технология | Қолданылуы |
|------------|------------|
| Kotlin | Android қосымшасын жазу |
| Room | Жергілікті деректер базасы (offline) |
| Firebase Authentication | Пайдаланушы тіркеу және кіру |
| Firebase Realtime Database | Бұлтта деректерді сақтау және оқу |
| Jetpack Compose / XML | UI құру |
